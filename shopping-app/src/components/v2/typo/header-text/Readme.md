# Header Text
Display a header text

### Properties
| Property | Type  | Default | Description |
| --- | --- | --- | --- |
| `text` | `string` |  | Display text |
| `icon` | `string`|   | Icon name |
| `iconForegroundColor` | `string`|  `#fff` | Icon foreground color |

### Usage
```
<HeaderText 
    text='Header text' 
    icon='home' 
    iconForegroundColor='#F1D674' 
/>
```
